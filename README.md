# ru Helper

> Удобный инструмент для установки, настройки и управления [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) и [tg-ws-proxy](https://github.com/Flowseal/tg-ws-proxy) — без ручной возни с bat-файлами и службами.

---

## ✨ Возможности

- 📥 Скачивание и установка zapret-discord-youtube и tg-ws-proxy прямо из последних релизов GitHub
- 🔍 Поиск рабочего альта
- ⚙️ Управление сервисами
- 🗑️ Удаление файлов программ одной кнопкой
- 🩺 Диагностика соединения — проверка доступности Discord, YouTube, Telegram, GitHub с пингом
- 🔔 Проверка версий при запуске — подсвечивает если доступно обновление

---

## 🚀 Быстрый старт

1. Скачай `ru_helper.exe` со [страницы релизов](../../releases/latest)
2. Запусти **от имени администратора**
3. Выбери пункт `[1]` — установить zapret
4. Выбери пункт `[3]` — найти рабочий альт
5. Зайди в `[5]` Управление сервисами → установи в автозапуск

---

## 📋 Требования

- Windows 10 / 11
- Запуск **от имени администратора**
- Интернет-соединение для скачивания

---

## ⚠️ Антивирус

zapret использует драйвер **WinDivert** для перехвата трафика — антивирусы могут ложно срабатывать на него. Добавь папку `%APPDATA%\ru_helper` в исключения антивируса.

---

## 📦 Что устанавливается

| Программа | Путь |
|-----------|------|
| zapret-discord-youtube | `%APPDATA%\ru_helper\zapret\` |
| tg-ws-proxy | `%APPDATA%\ru_helper\tg-ws-proxy\` |
